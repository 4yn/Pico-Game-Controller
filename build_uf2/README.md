# Pico-Game-Controller

Built uf2 file for quick flashing. CMake should copy to here upon build.

Quick how-to flash:

1. Hold down Pi Pico button when plugging it in.
2. Drag Pico_Game_Controller.uf2 into the 'drive' RPI-RP2
3. Done!
